# Projeto Tabuada em JavaScript

Descrição
-

Este projeto consiste em um gerador de tabuadas desenvolvido com HTML, CSS e JavaScript. Ao clicar no botão da interface, o sistema gera automaticamente as tabuadas do número 1 até o 100, exibindo as multiplicações de 1 a 10 para cada número.

O objetivo principal é praticar lógica de programação, estrutura de repetição e manipulação do DOM.

Objetivo
-

1. Aplicar conceitos de lógica com laços de repetição

2. Trabalhar com manipulação dinâmica de conteúdo na página

3. Organizar elementos visualmente utilizando CSS

4. Estruturar código de forma clara e funcional

Funcionamento
--

Quando o usuário clica no botão "Gerar Tabuada", a função responsável é executada.

O sistema:

- Seleciona o elemento da página onde as tabuadas serão exibidas

- Cria uma variável vazia para armazenar todo o conteúdo gerado

- Utiliza um laço de repetição para percorrer os números de 1 até 100

- Para cada número, executa um segundo laço que realiza as multiplicações de 1 até 10

- Armazena cada resultado dinamicamente

- Insere todo o conteúdo gerado na página de uma única vez

Conceitos Aplicados
--

- Funções em JavaScript

- Estrutura de repetição for

- Manipulação do DOM

- Template Strings

- Uso de let e const

- Estruturação de layout com Flexbox

Estrutura Lógica
--

O projeto utiliza dois laços de repetição:

O primeiro controla qual tabuada está sendo criada (1 a 100)

O segundo controla as multiplicações internas (1 a 10)

Essa estrutura permite a geração automática de 100 tabuadas completas de forma organizada.

Resultado
-

O sistema exibe as tabuadas organizadas em colunas, com separação visual e estrutura centralizada, proporcionando melhor legibilidade e organização.
